## рџ“– Project Overview

**Simon Game** is a browser-based memory game inspired by the classic Simon electronic game. The objective is to remember and repeat a sequence of colors generated by the system.

Each round adds a new color to the sequence, making the game progressively more challenging. The player must carefully observe and repeat the pattern correctly to move to the next level.

This project demonstrates important frontend development concepts such as **DOM manipulation, event handling, random sequence generation, animations, and interactive game logic using JavaScript**.
